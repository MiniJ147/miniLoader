# MiniLoader

## What is this about?

Simple live reloader for go projects, No Config file, No Bloat, No nothing besides one command.  

## Use

### Installation

```
go install github.com/MiniJ147/miniloader@latest
```

### Running
```
(windows)
miniloader {path to main.go file} 

(linux)
./miniloader {path to main.go file} 
```
